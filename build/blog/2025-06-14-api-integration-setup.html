API Client Setup

Quick setup for a robust API client:

const apiClient = {
  async get(endpoint) {
    const response = await fetch(`/api/${endpoint}`);
    if (!response.ok) throw new Error(`API Error: ${response.status}`);
    return response.json();
  }
};
